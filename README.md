# 🏠 Analisis Data Properti Rumah

## 📌 Deskripsi Project
Project ini bertujuan untuk menganalisis faktor-faktor yang mempengaruhi harga properti rumah berdasarkan dataset penjualan properti. Analisis ini digunakan untuk memahami pola harga, distribusi data, serta hubungan antar variabel yang dapat membantu dalam pengambilan keputusan bisnis.

---

## 🎯 Tujuan Analisis
- Mengetahui distribusi harga properti
- Membandingkan nilai mean dan median untuk memahami skewness
- Menganalisis hubungan antara luas properti dengan harga
- Mengidentifikasi faktor-faktor yang mempengaruhi harga rumah
- Memberikan insight untuk strategi pricing

---

## 📊 Dataset
Dataset berisi 178 data properti dengan beberapa fitur utama:
- Luas Properti
- Jumlah Kamar Tidur & Mandi
- Fasilitas (Kolam Renang, Gym)
- Kota
- Jarak ke Stasiun
- Harga Properti

---

## 🛠️ Tools & Library
- Python
- Pandas
- Matplotlib / Seaborn
- Google Colab

---

## 🔍 Insight Utama
- Distribusi harga cenderung tidak normal (terdapat skewness)
- Mean lebih tinggi dari median → indikasi adanya outlier harga tinggi
- Luas properti memiliki korelasi positif terhadap harga
- Lokasi dan jarak ke transportasi mempengaruhi nilai properti

---

## 📈 Analisis yang Dilakukan
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualisasi Data
- Analisis Korelasi

## 📌 Kesimpulan
Analisis menunjukkan bahwa harga properti dipengaruhi oleh berbagai faktor seperti luas bangunan, lokasi, dan fasilitas. Pemahaman distribusi data sangat penting agar tidak salah dalam menentukan strategi harga.
